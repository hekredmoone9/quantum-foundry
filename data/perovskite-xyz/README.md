# Perovskite-XYZ DFT Simulation Dataset

## Description
This dataset contains density functional theory (DFT) simulation results for the novel 2D perovskite material "Perovskite-XYZ". The simulations were performed to evaluate the material's electronic structure and optical properties for next-generation solar cell applications.

## Contents
- `params.json`: Simulation parameters (temperature, pressure, k-point grid, etc.)
- `results.json`: Key results including bandgap, absorption spectrum peaks, and total energy.

## Simulation Details
- **Method**: DFT with PBE functional
- **Software**: Quantum ESPRESSO v7.0
- **Temperature**: 300 K
- **Pressure**: 1 atm
- **k-point grid**: 12x12x1
- **Energy cutoff**: 80 Ry

## Usage
This data is intended for reproducibility and further analysis within the quantum-foundry project. Please cite appropriately.
